# ⏳ Digital Time Capsule

A full-stack MERN application that allows users to create, store, and preserve digital memories, messages, and files to be unlocked at a specified date in the future.

---

## 🚀 Features

* **Create Capsules:** Store text, notes, or media links intended for your future self or others.
* **Time Lock:** Capsules remain sealed until their scheduled unlock date/time.
* **User Authentication:** Secure access to manage and view your personal time capsules.
* **Dashboard View:** Easily view locked and unlocked capsules at a glance.

---
## 🛠️ Tech Stack

* **Frontend:** React, HTML5, CSS3, JavaScript (ES6+)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB / JSON Storage
* **Environment Management:** `dotenv`
